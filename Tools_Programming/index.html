For my tools module in university i was tasked in taking a basic world editor inspired by the World of Warcraft Editor and adding functionality to aid artists and designers. I added picking functionality, camera functionality and for my primary addition I created a gizmo. The gizmo lets the user move the object(s) in linear directions or planar directions by selecting different parts of he gizmo and dragging. the user can also rotate the object in any direction and scale in any direction or in all directions with the scroll wheel.
